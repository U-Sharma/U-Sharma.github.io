Sanskrit (संस्कृत, IAST:Saṁskṛta) is the primary language of yoga, and of many other disciplines from Southeast Asia. It is a remarkably algorithmic langauge, and yet is beautifully poetic. In the modern times, by which I mean the last few thoudand years, Sanskrit has been written in Devanagari (Devanāgarī) script. The last few decades have seen an increasing use of a romanized script for Sanskrit, called IAST. You can read about it on the Wikipedia page

Devanāgarī Keyboard Layout

The standard Devanāgarī keyboard is hard to use with keyboards that have roman letters printed on them. I have designed a IAST style keyboard to type Devanāgarī.
